웹 개발을 할때 귀찮은 돔 관리와 상태값 업데이트 들을 최소한으로 하고
오직 기능 개발 사용자 인터페이스를 집중할수 있도록 프론트엔드 라이브러리가 필요함

프론트엔드 라이브러리 삼대장

angular
여러 기능들이 이미 내장 되어있음, 메인으로 사용하는 개발자들이 많지 않아서 인지도는 쌓아가는중
typeScript 기본 사용
(http client 등)

vue
웹팩 같은 모듈 번들러가 없어도 cdn으로 불러와서 사용가능
(cdn - 음원, 영상등 10메가 이상의 대용량 파일을 안정적으로 제공하기 위해 구상된 기술,
구상과정에서 자주 사용되는 병목현상을 해결할 수 있음을 확인되었다.)
cdn으로 불러온다는 것은 html 태그 중 script 태그로 불러온다는 것을 의미함
html을 템플릿처럼 그대로 사용가능

react

component 라는 개념이 집중되어있는 라이브러리
뷰만 신경쓰고 나머지 기능은 서드파티라이브러리에서 담당(redux, react-router)
공식 라이브러리 개념이 없고 여러 라이브러리를 사용해야함
